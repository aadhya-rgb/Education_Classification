# Education Level Classification (UN SDG Aligned)

# Description
This project predicts education levels based on socio-economic indicators using machine learning. The analysis aligns with UN Sustainable Development Goal 4: Quality Education, aiming to support data-driven insights for improving educational outcomes.

# Dataset

Features include demographic and socio-economic variables relevant to education level prediction

Target: Education level classification

Purpose: Identify factors influencing education attainment and predict student/community education levels

Key Features & Analysis

Explored data distributions and feature correlations

Performed preprocessing: handling categorical variables, scaling, and encoding

Visualized patterns using Matplotlib and Seaborn

Machine Learning Models

Classification models: Random Forest, Logistic Regression, and other classical ML models

Evaluated performance using accuracy, confusion matrix, and other metrics

Applied feature selection and hyperparameter tuning for optimal results

# Insights

Identified socio-economic indicators most predictive of education level

Demonstrated interpretable machine learning pipeline for decision support

# Technologies Used
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
